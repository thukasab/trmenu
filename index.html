MENU2 レジスタンストレーニング

01_スクワット

  • 負荷:50kg /
  • rep:8rep /
  • set:3set

脚は肩幅か少し広めのスタンス。つま先は軽く外向きにしつま先と膝の向きを揃える。立ち上がり動作でジャンプして着地時に脚を閉じて着地する。

参考動画 01:35くらいから

02_ジャンプスクワット

  • 負荷:20kg /
  • rep:10rep /
  • set:3set

脚は肩幅より広めのワイドスタンス。つま先は軽く外向きにしつま先と膝の向きを揃える。

参考動画 03:05くらいから

03_ヒップスラウト

  • 負荷:70kg /
  • rep:10rep /
  • set:3set

バーの位置は恥骨あたり、ベンチの位置は肩甲骨あたりに来るように調整してセットする。つま先と膝の向きを揃え、腹圧を高め大殿筋の収縮を意識する。
